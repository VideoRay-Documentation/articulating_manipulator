Software Management

The Articulating Manipulator includes software that modifies VideoRay Cockpit to recognize the Articulating Manipulator, Articulating Cutter or standard VideoRay manipulator. You can switch the active configuration as often as necessary to match the manipulator in use, but only one configuration may be active at any time. The configurations and instructions for selecting the desired configuration are included below.

VideoRay Cockpit software must be stopped to change configurations.

  1. Articulating Manipulator - use this configuration when using the Articulating Manipulator with any jaw except the cutter. To select this configuration, double click on the Use Articulating Manipulator shortcut on the desktop.

  2. Articulating Cutter - use this configuration when using the Articulating Manipulator with the cutter. To select this configuration, double click on the Use Articulating Cutter shortcut on the desktop.

  3. Standard Manipulator - use this configuration when using the standard manipualtor. This is the default configuration installed on all Pro 4s. To select this configuration, double click on the Use Standard Manipulator shortcut on the desktop.
