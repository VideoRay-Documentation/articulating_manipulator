Software Installation

To install the Articulating Manipulator software, follow these instructions:

  1. Use the flash drive included with the Articulating Manipulator, or download the software from VideoRay's software download page.
  2. Double click articulating_manipulator.exe. This will copy the files to your local hard drive.
  3. Navigate to C:\VideoRay\Installs\VideoRay\articulating_manipulator\ and double click install.bat.
