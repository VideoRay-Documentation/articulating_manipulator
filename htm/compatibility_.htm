Component Compatibility

The Articulating Manipulator is compatible with the Pro 4.

Standard and BlueView Center Float

With just the standard float and no ballast, the ROV can handle a maximum of 1.14 lbs of extra weight. With the BlueView float and no ballast, the ROV can handle a maximum of 1.75 lbs of extra weight. The BlueView sonar takes up all the extra weight capacity here, so if you have the sonar attached, there is no room for any other accessories. If heavy accessories are equipped to the Pro 4 the BlueView Float should be used and the rear ballast should be detached. If the standard is used with heavy accessories it is possible that the ROV could sink.

Communications

The Articulating Manipulator uses the Cockpit software to communicate with the Videoray Pro 4. See the software guide for more details.

Mounting

The Articulating Manipulator can be mounted by attaching it to the centre float and mount plate. See the equipment guide.
