Installation

Software Management

Operating the Articulating Manipulator requires an installation of VideoRay Cockpit 2015 which can be acquired from the VideoRay flash drive accompanying the kit. The Articulating Manipulator will not function on any previous versions of the Cockpit software.

1) Download the Cockpit 2015

2) Run Cockpit 2015 Installer Package

The Articulating Manipulator also requires a specific device file in order to operate properly with our Cockpit software.

1) Copy the STANDARD_MANIP and STANDARD_INUKTUN_MANIP folders to your desktop.

2) In the STANDARD_INUKTUN_MANIP folder please run the AccessoryConfiguration.exe.

a. Running the AccessoryConfiguration.exe will install the device file to appropriate location.

3) Upon opening Cockpit 2015, you will find that the Cockpit Instrument for the Manipulator is now configured for the Articulating Manipulator.

4) Plug the Articulating Manipulator into the Pro 4 ROV using the Pro 4 cable and verify operations.

Inserting and Removing the Articulating Manipulator Jaw

1) Connect the Articulating Manipulator Jaw to the body by inserting the body into the jaw's brass piece.

2) To attach the Articulating Manipulator Jaw use the 3mm socket wrench located in the tool kit to screw in the brass cap screws.

3) Once the jaw is attached use the Phillips screw driver to put the screws in the Articulating Manipulator head.

4) To remove the jaw simply unscrew the screws holding the jaw, followed by the cap screws. Then remove the head.

Mounting the Articulating Manipulator

1) To mount the Articulating Manipulator on the VideoRay Pro 4, place the Articulating Manipulator Body on the underside of either the BlueView or standard center float.

2) Position the Articulating Manipulator Body so that the screw holes align to the center float, use the screws to connect the components.

3) Align the body and center float in the same way with the BlueView Skid Bottom Plate or the Pro 4 Skid Centre Plate.

4) Use the Phillips screw driver to screw in the jaw head screws and connect all the components.

If additional information is needed on VideoRay Pro 4 parts, please see the pro 4 equipment guide.

Additional Details

The images above are guides for installation. The first image uses the standard Pro 4 skid plate and center float. The second image substitutes these for a BlueView Skid plate and float. The difference between the two assembly sets is that the standard configuration has a maximum weight limit of 1.14 lbs of extra equipment. With the BlueView Skid and float that changes to a maximum of 1.75 lbs. If the Articulating Manipulator is mounted with the BlueView Sonar the brass Rear Ballast piece should be removed from the back of the manipulator body. This reduces the ROV's weight, and improves the balance of the ROV with the added accessories. For more information on individual parts see the equipment guide.
