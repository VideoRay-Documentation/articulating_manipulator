Installation

The image above is a guide for installation. In order to install the Articulating Manipulator to the VideoRay Pro 4, place the center float on top of the BlueView Skid Bottom Plate or the Pro 4 Skid Centre Plate, seen here. Place the Articulating Manipulator over the center float. Position the Articulating Manipulator so that it can be screwed into the center float using the 4X M4X40 and M4X25 SHCS screws. If additional information is needed on VideoRay Pro 4 parts, please see the pro 4 equipment guide.
