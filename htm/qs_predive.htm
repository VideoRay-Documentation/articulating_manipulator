Before attempting to use the Articulating Manipulator, the user should install the necessary software, which should be included on the VideoRay flash drive accompanying the kit. Alternatively the software can be acquired on the VideoRay website. For more details on this, see the software guide. Mount the preferred jaw on the Articulating Manipulator body and perform a functions test, check to make sure all components are secured tightly. See the operations guide for more details on how to mount the manipulator.
